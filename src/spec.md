# Specification

## Summary
**Goal:** Simplify the product browsing experience by removing search and filter controls and replacing them with a direct "Browse All Products" link.

**Planned changes:**
- Remove search input field from ProductCatalogSection
- Remove category filter buttons from ProductCatalogSection
- Add a horizontal bar below the products grid displaying "Browse All Products" with a link to the catalog

**User-visible outcome:** Users will see a cleaner product section without search/filter controls, and can click "Browse All Products" to access the full catalog.
